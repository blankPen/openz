核心页面 · 4 屏并排
① 空对话态 home.html
冷启动 / 清空后首页。5 区域：顶部导航 / 欢迎招呼 / 3 张推荐卡 / 横滑工具栏 / 输入区 + AI 水印。
② 模型切换 model-switch.html
点击首页模型 pill 弹出 bottom sheet。三段分组：基础模型 / 推理模式 / Agent 人格。
③ 附件面板 attachment.html
点击 + 按钮弹出。4 入口：本地图片 / 本地文件 / 拍照 / 引用回复 + 最近使用 + 拖拽提示。
④ 对话进行态 conversation.html
4 种气泡：用户 / AI 基础 / AI 思考 / 工具调用。Markdown 渲染、代码块、引用 @、4 项操作区。
侧边设置菜单 · 浅 / 深主题切换
⑤ 设置菜单 · 浅色 settings.html
点击首页左上角汉堡按钮 或 从左边缘右滑（≥80px）唤起。4 段菜单：通用 / 智能助手 / 账户 / 其他。
⑤ 设置菜单 · 深色 settings.html?theme=dark
点「深色」立即全 app 切换：背景、抽屉、菜单项、灰化首页、Home indicator 一起翻转。状态写 localStorage，跨页保留。
